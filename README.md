# 🖥️ Programme C# en console

Application console en **C# / .NET** regroupant plusieurs programmes interactifs accessibles depuis un menu principal.

---

## 🎯 Objectif

Ce projet vise à pratiquer :

* Tableaux et matrices
* Entrées utilisateur
* Navigation via menus (`switch / case`)
* Logique de programmation

---

## 📁 Structure du projet

```bash
dotNet/
├── Program.cs
└── README.md
```

---

## 🚀 Lancement

```bash
dotnet build
dotnet run
```

> Requiert **.NET 6.0** ou supérieur.

---


---

## 📌 Programme 1 — Exercices de base

* Somme de deux nombres
* Division (avec vérification)
* Permutation de valeurs
* Affichage d’un triangle
* Conversion en binaire

---

## 📌 Programme 2 — Tableau

* Initialisation du tableau :

  * Saisie manuelle
  * Génération aléatoire

* Affichage des éléments

* Tri du tableau

* Recherche du minimum et du maximum

* Menu interactif avec retour possible


---

## 📌 Programme 3 — Matrices

* Saisie ou génération aléatoire
* Affichage
* Inversion des colonnes

---

## 📌 Programme 4 — Recherche de caractère

* Saisie de chaînes (`stop` pour arrêter)
* Stockage en `List<char[]>`
* Recherche d’un caractère avec affichage des positions
* Sortie avec la touche `ESC`

---

## 🛠️ Concepts utilisés

* `switch / case`
* Boucles (`for`, `while`)
* `List<char[]>`, `int[,]`
* `Console.ReadKey()`
* Conversions de types
